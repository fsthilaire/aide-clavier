Ajouter une disposition

Important

Les ordinateurs du Cégep utilisent Windows 10.

Windows 10​

  1. Appuyez sur le menu Démarrer et sélectionnez l'item Paramètres.

  2. Dans la fenêtre des Paramètres de Windows, sélectionnez l'item Heure et langue.

  3. Sélectionnez le menu Langue à gauche.

  4. Dans la section Langues par défaut, sélectionnez la langue Français (Canada) et appuyez sur le bouton Options.

  5. Dans la section Claviers, ce sont les dispositions actuellement installées dans Windows. Le bouton + permet d'ajouter une nouvelle disposition.

  6. Sélectionnez dans la liste la nouvelle disposition.

  7. La nouvelle disposition est maintenant installée.

Windows 11​

  1. Appuyez sur le menu Démarrer et sélectionnez l'item Paramètres.

  2. Dans la fenêtre des Paramètres de Windows et sélectionnez l'item Heure et langues dans le menu à gauche. Ensuite, sélectionnez l'item Langue & région.

  3. Dans la section Langue, sélectionnez la langue Français (Canada), appuyez sur le bouton ... et sélectionnez ... Options linguistiques.

  4. Dans la section Claviers, ce sont les dispositions actuellement installées dans Windows. Le bouton Ajouter un clavier permet d'ajouter une nouvelle disposition.

  5. Sélectionnez dans la liste la nouvelle disposition.

  6. La nouvelle disposition est maintenant installée.
